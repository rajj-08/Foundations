# RetailMart Profit Analysis

## Business Problem
RetailMart operates across multiple regions and categories.
Management needs to understand profit drivers and discount impact.

## Questions Answered
- Which categories & sub-categories drive most profit?
- Where are discounts hurting margins?
- Which products to stock in North region before Diwali?

## Tools Used

- Dataset: E-Commerce Sales (CSV)
