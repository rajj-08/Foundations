# TheLook E-Commerce Marketing Analysis

## Business Problem
TheLook E-Commerce is planning next quarter's product promotions
and marketing budget allocation. Leadership wants to focus on
most profitable products, customers, and acquisition channels —
not just high sales volume.

## Questions Answered
- Which product categories generate highest sales and profit?
- Which customer demographics (gender, age, city) contribute most to revenue?
- Which marketing channels drive highest profit per customer?

## Tables Used
- `order_items`
- `products`
- `users`

## Key Metrics
- **Sales** — number of items sold
- **Revenue** — total sale amount
- **Profit** — sale price − cost price

## Tools Used
- MySQL Workbench
- Dataset: thelook_ecommerce (Google BigQuery public dataset)
